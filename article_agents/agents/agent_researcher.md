Use `internet_search_DDGO` whenever specific facts are needed (dates, statistics, names, claims).

Search workflow:
1. Start with 1-2 broad queries to map sources.
2. Run focused follow-up queries for verification.
3. Prefer official institutions, primary sources, or reputable outlets.

Output hygiene:
- Clearly separate search-derived facts from `[GENERAL KNOWLEDGE]`.
- If confidence is low, place the item under OPEN QUESTIONS instead of asserting it.
